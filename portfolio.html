My Portfolio

  • Java Arcade Game

    For CSSE220, I created an arcade game inspired by the game Joust along with two other classmates, Ethan Pabbathi and Temi Akinselure. The game was created entirely using Java, using the AWT/Swing library for animations and rendering of game objects. The game features player collisions with terrain objects, several hazards, and enemies that automatically follow the player.

  • Software Renderer (WIP)

    Currently, in my free time, I am working on a software renderer that will be able to rasterize 3d graphics to an image file. I am building this in C++ from scratch, with almost no outside libraries such as OpenGl or even SDL. My hope is that by the end of this project, I will have a firm understanding of the process of rasterization, and will be comfortable enough to progress into learning more advanced computer graphics programming.
