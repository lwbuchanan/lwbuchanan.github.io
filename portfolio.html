My Portfolio

  • Java Arcade Game

    For CSSE220, I created an arcade game inspired by the game Joust along with two other classmates, Ethan Pabbathi and Temi Akinselure. The game was created entirely using Java, using the AWT library for animations and rendering of game objects. The game features player collisions with terrain objects, several hazards, and enemies that automatically follow the player.
